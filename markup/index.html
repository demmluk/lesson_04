Какими бывают селекторы?

Селекторы - это части кода CSS (Cascading Style Sheets), которые позволяют выбирать и стилизовать HTML элементы на странице. Селекторы указывают браузеру, какие элементы нужно изменить или стилизовать, используя правила CSS.

Селектор по названию элемента:

Выбирает все элементы определенного типа. Например, p выберет все абзацы "p" на странице.

Селектор по id:

Выбирает уникальный элемент по его идентификатору. Например, #header выберет элемент с id="header".

Селектор по классу:

Выбирает все элементы с определенным классом. Например, .button выберет все элементы с class="button".

Селектор по нескольким классам:

Выбирает элементы, которые имеют одновременно несколько классов. Например, .violet.bold выберет элементы с классами class="violet bold".

Селектор по вложенности:

Выбирает элементы, которые находятся внутри другого элемента. Например, ul li выберет все элементы li, которые находятся внутри ul.
